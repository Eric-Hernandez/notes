Intra-page linking

How do I link to a specific part of a page?

You need two components:

  • a link
  • a target for the link

Link

It's just like any other link, except we refer to an "id" value within our page.

<a href="#cats">cats</a>

The value should:

  • begin with a hashtag (#)
  • be lowercase
  • not include spaces
  • be unique

id

id is a global attribute. It can be used with any tag.

Values must be unique.

Using id

Using id with a tag gives our link a target.

<section id="cats"></section>

The value should:

  • not begin with a hashtag
  • be identical to the value set in the link
  • be unique (do not use the same ID twice)
