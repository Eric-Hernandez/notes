HTML Structure

Giving the web meaning

  • structural tags describe portions of a web page
  • helpful when using CSS, Javascript

<header>

  • represents introductory content or navigation
  • may contain a logo

<nav>

  • used for site navigation
  • may be used in header, footer or independently
  • generally contains a list (<ul>)

<main>

  • describes the portion containing actual, unique content
  • may contain <section>, <article>, <aside>

<section>

  • organizes generic portions of a document
  • should generally contain a h2-h6 element
  • may contain <article>, <aside>

<article>

  • organizes content within a section
  • "a self-contained composition"
    • forum post
    • magazine or newspaper article
    • blog entry
    • user-submitted comment
    • interactive widget or gadget
    • any other independent item of content

<aside>

  • represents content related to, but possibly separate from the content
  • a sidebar or insert
    • definitions
    • author bio
    • related links
    • data tables
    • images/captions
    • figures

<footer>

  • may contain:
    • author information
    • copyright
    • secondary navigation
