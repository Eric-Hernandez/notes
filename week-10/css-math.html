Converting Between Units

Pixels are fixed units, which are great to work with if we want 'pixel-perfect' designs. They are not helpful when we want to create flexible, responsive designs.

How do we convert our designs from pixels to relative measurements? With a little bit of math.

By default, (most) browsers have a default font size of 16 pixels. What if a user changes these settings? It is okay — our font size is still proportional to the new, user-defined font size.

px to em

Formula

size in pixels / parent size in pixels = ems

Examples

12px / 16px = .75em

36px / 16px = 2.25em

48px / 16px = 3em

When using ems, the parent size is based on the immediate parent, not html.

If a you are calculating the font size for an element whose parent also has a font size, you take into acount the parent's font size in your calculation.

px to percentage

Formula

(size in pixels / parent size in pixels) * 100 = percentage

Examples

(12px / 16px) * 100 = 75%

(36px / 16px) * 100 = 225%

(48px / 16px) * 100 = 300%

When using percentage, the parent size is based on the immediate parent, not html.

If a you are calculating the font size for an element whose parent also has a font size, you take into acount the parent's font size in your calculation.

px to rem

Formula

size in pixels / html size in pixels = root ems

Examples

12px / 16px = .75rem

36px / 16px = 2.25rem

48px / 16px = 3rem

When using root ems, the parent size is based on the font-size property set on html.

Tips

Since you're just starting out, it is okay to use pixels. You can always convert your units in a later iteration of your web site design.

Become proficient in using fixed design practices, then try to implement fluid design practices.

When making conversions, be sure you leave a comment in your CSS as a reminder of your calculation. For instance:

font-size: 1.875em; /* 30px / 16px */
