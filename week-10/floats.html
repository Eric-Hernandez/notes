Floats

Floating Elements

The float property allows us to wrap text around images and/or other elements.

The property accepts three values:

  • left
  • right
  • none

Whenever float is used the element's display value becomes block.

Clearing Floats

Remember how we talked about CSS 2.0 being weird and buggy?

Clearing floats is a pretty good example of how web developers had to find hacks to get their code right.

There are a few solutions, all of which may work in different situations.

Try what works. If it does, stick with it.

clear property

Sometimes we need content within the normal flow to appear after a floated element, as if the content were starting on a new line.

To end a float and re-establish the normal flow, we use the clear property.

If we're trying to move past:

  • left-floated elements, use clear: left;
  • right-floated elements, use clear: right;
  • left- and/or right-floated elements, use clear: both;

The clear property also takes the value none.

:after pseudoclass

If you want an element to contain all elements inside of it, use the clear property on an element which uses the :after pseudoclass.

For example:

section:after {
  content: "";
  display: block;
  clear: both;
}

.clearfix class

Another way of enclosing a floated element within its container:

.clearfix {
  overflow: auto;
  zoom: 1;
}

Then add class="clearfix" to the containing element in HTML.

This is a pretty standard "hack" which still works in older browsers. It's not the nicest solution, but sometimes you go with what works.
