Absolute positioning

Not Positioned

Positioned

An element with its position set as absolute acts like a fixed element, but relative to the nearest positioned ancestor.

If the ancestor is static, the element will be positioned in relation to the document body.

To make this work:

  • the parent element must have its position designated as relative
  • we need to specify a horizontal (x-axis) property (left, right) and a vertical (y-axis) property (top, bottom) of the element to be moved.
